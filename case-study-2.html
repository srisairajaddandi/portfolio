Case Study — 01

AUTOMATIC DETECTION OF STUDENT ENGAGEMENT DURING ONLINE CLASSES

AI-powered real-time engagement monitoring using facial landmark detection & behavioral analytics

Overview

An AI-powered solution that evaluates student attentiveness during online learning sessions using computer vision and facial landmark detection. The system analyzes facial expressions and behavioral cues captured via webcam — monitoring eye movements, yawning patterns, and head posture to determine real-time engagement levels.

Background

Rapid growth of online education makes continuous student monitoring nearly impossible for instructors. Traditional classrooms allow visual attentiveness checks — virtual environments cannot replicate this.

Students can become distracted, drowsy, or disengaged without educators being aware. Existing platforms focus on content delivery but lack intelligent behavioral monitoring mechanisms.

Absence of engagement analytics limits teaching effectiveness and delays timely intervention.

Solution

An intelligent engagement monitoring framework using AI + Computer Vision
Eye Tracking
Eye Aspect Ratio (EAR) measures eyelid distance to detect closure, blinking, and drowsiness patterns over time.
Yawn Detection
Mouth Aspect Ratio (MAR) quantifies mouth openness to identify yawning as an indicator of fatigue.
Head Pose
Face Angle (FA) calculates head orientation relative to the camera to detect distraction or loss of focus.
Visual Analytics
Flask web dashboard with real-time pie charts and graphs displaying engagement metrics per session.

Gallery

Outcome

User ↔ Camera ↔ Detection actors interact through webcam live track, process frame, detect face, yawn/eye/pose operations. User → Application → Camera → FaceRecog → Face Landmark Model. Features extracted, predicted (YWAN, POSE, EYE), compared and visualized.

Three classes: server.py (capture/live frames), severf.py (videocapture/predict), facemodel.py (eye/yawn/pose methods). Webcam feed → face detected? → Get landmarks → Threshold check → Yawn/eye close detected → Loop or quit.

Reflection

The Automatic Student Engagement Monitoring System effectively bridges the gap between traditional and online education by providing intelligent real-time engagement analysis.
Accurately detects signs of distraction, fatigue, and disengagement using facial landmark analysis
Real-time EAR, MAR, and Face Angle metrics with color-coded web dashboard alerts
Demonstrates practical application of AI, ML, and Computer Vision in education technology.
Scalable and extensible to institutions, corporate training, and adaptive learning systems.

Future Scope: Emotion recognition · Adaptive learning · Voice analysis · Cloud deployment · Mobile app
